title: Offer the picnic sports ground for community rental by tender
composed: 2026-06-21

That, if no existing extension of use has been granted, the field be offered
to the community for a rental fee, by tender, the fee yet to be determined;
and that if an extension has been granted, it be honoured with payment
requested up front.
